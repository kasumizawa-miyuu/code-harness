用户指南

什么是 code-harness？

一个轻量级 Coding Agent Harness，通过反馈循环驱动主循环。它读取你的任务，调用 LLM，执行动作（读/写文件、运行命令），并根据验证结果自适应调整。

使用工作台

  • 在左侧边栏配置 LLM Provider、API Base URL 和 Model
  • 通过侧边栏设置 API Key（存储在浏览器中）
  • 在输入框中输入任务
  • 点击 Run Task 或按 Ctrl+Enter 执行
  • 开启 Process Logs 查看 Agent 的推理过程

Docker 本地文件访问

docker run -v $(pwd):/workspace -w /workspace -p 3000:3000 code-harness

将当前目录挂载到容器内的 /workspace，Agent 可以读写该目录中的文件。

CLI 直接使用

git clone https://github.com/kasumizawa-miyuu/code-harness.git cd code-harness npm install npm run build npm link harness configure harness key update harness run "your task"
