User Guide

What is code-harness?

A lightweight Coding Agent Harness with a feedback-loop-driven main loop. It reads your task, calls an LLM, executes actions (read/write files, run commands), and adapts based on verification results.

Using the WebUI

  • Configure your LLM provider, API base URL, and model in the sidebar
  • Set your API Key via the sidebar (stored securely in OS keychain)
  • Type a task and click Run Task
  • Toggle Show process logs to see the agent's reasoning

Working with Local Files (Docker)

To give the WebUI access to your local files, run with Docker volume mount:

docker run -v $(pwd):/workspace -w /workspace -p 3000:3000 code-harness

This mounts your current directory into /workspace inside the container. The agent can read, write, and modify files in this directory.

Using the CLI (Direct Access)

Clone the repository to use the CLI directly on your local files:

git clone https://github.com/kasumizawa-miyuu/code-harness.git cd code-harness npm install npm run build npm link harness configure harness key update harness run "your task"

CLI works directly on your local filesystem — no Docker needed.

Remote Deployment

When deployed remotely (Render, etc.), the WebUI operates on the server's filesystem. Use Docker with volume mounts to connect your local files.
